# Picbook

Picbook是一个单文件、纯静态的本地图画书编辑工具。当前编辑器入口位于：

'editor.html`

仓库根目录的 `index.html` 会自动跳转到编辑器，方便用 GitHub Pages 展示。

## 隐私与本地处理

本工具不依赖服务器后端、数据库或云端处理服务。用户导入的照片、字体、背景图、项目文件等内容均在浏览器本地处理。工具不会主动上传、传输、收集或保存用户文件到任何服务器，也不包含用户账号系统或远程隐私数据分析功能。

浏览器自动记忆仅保存轻量偏好设置；书本页面、照片、文字等项目内容请通过 `文件 > 保存文件` 保存。如需重新编辑项目或留存项目，请自行妥善保存项目文件、原始照片、字体文件和导出的印刷文件。

## GitHub Pages

推荐 Pages 设置：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

启用后访问仓库的 GitHub Pages URL 即可打开编辑器。
